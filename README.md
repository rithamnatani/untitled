# Story Vault

This vault is set up for a light novel that can later be adapted into a manhwa.

Use it as plain Markdown in Zed, or open the folder as an Obsidian vault for wikilinks, backlinks, and graph view.

## Structure

- `00-Home/` - entry notes and project dashboard
- `01-Story/` - premise, themes, plot, arcs, chapter planning
- `02-World/` - setting, factions, rules, locations
- `03-Characters/` - one file per character
- `04-Chapters/` - chapter drafts and outlines
- `05-Art-Refs/` - visual references and manhwa adaptation notes
- `_templates/` - reusable note stubs
